## App Details

**App name:** CoverGuard
**Short description (80 chars max):**
Check property flood, fire, earthquake & crime risks — find active carriers.

**Full description (4000 chars max):**
CoverGuard is the smart way to understand property insurance risks before you bid on a home.

Search any US property address to instantly see:

RISK INTELLIGENCE
- Flood risk: FEMA flood zones, SFHA designation, base flood elevation, historical claims
- Fire risk: Cal Fire FHSZ severity zones, USFS Wildland-Urban Interface data
- Earthquake risk: USGS seismic hazard data, ASCE 7-22 spectral acceleration
- Wind risk: ASCE 7 design wind speeds, NOAA hurricane surge zones
- Crime risk: FBI Crime Data Explorer statistics by jurisdiction

INSURANCE INTELLIGENCE
- See which insurance carriers are actively writing and binding policies for the property
- Understand insurability challenges based on the risk profile
- Get estimated insurance costs before making an offer
- Request binding quotes directly from active carriers

BUILT FOR REAL ESTATE PROFESSIONALS
- Agent dashboard with client management
- Compare up to 3 properties side-by-side
- Save properties and add notes
- Analytics on your search activity and risk distribution
- Generate comprehensive property reports

BUILT FOR HOME BUYERS
- Simple property search by address or ZIP code
- Clear risk summaries with color-coded severity levels
- Save and track properties you're interested in
- Request insurance quotes with one tap

CoverGuard pulls data from authoritative public sources including FEMA, USGS, NOAA, Cal Fire, USFS, and the FBI Crime Data Explorer to give you the most accurate risk picture possible.

Know before you bid. Download CoverGuard today.
